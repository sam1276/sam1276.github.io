Functions

What is a Function in python

In Python, a function is a reusable block of code that performs a specific task or set of tasks. Functions allow you to encapsulate a sequence of statements into a single unit, which can be called and executed elsewhere in your code. They play a fundamental role in structuring and organizing Python programs, making them more readable, maintainable, and modular.

Types of Functions

Built-in Functions
User-Defined Functions

Built-in Functions

Built-in functions in Python are functions that are pre-defined in the Python language and are always available for use without requiring you to write their implementations. These functions are part of the Python standard library and serve a wide range of purposes. They cover various areas of programming, such as mathematics, string manipulation, data manipulation, input/output, and more.

Here are some examples of common built-in functions in Python:

`len()`: Returns the length (number of items) of an iterable object like a string, list, or tuple.

length = len("Python")

`input()`: Accepts user input from the keyboard as a string.

name = input("Enter your name: ")

User-Defined Functions

User-defined functions in Python are functions that programmers create to perform specific tasks or encapsulate a block of code for reuse within their programs. Unlike built-in functions, which are provided by the Python language, user-defined functions are defined by the programmer. These functions allow you to modularize your code, making it more organized, readable, and maintainable.

Here are the key characteristics of user-defined functions in Python:

1] Function Definition: To create a user-defined function, you use the `def` keyword, followed by the function name, a set of parentheses containing zero or more parameters (input values), and a colon. The function's code block is indented and contains the instructions for what the function does.

def my_function(parameter1, parameter2):
# Function body
# Code to perform specific tasks

2] Function Call: To execute a user-defined function and utilize its functionality, you simply call it by its name, passing the required arguments (if any) within the parentheses.

result = my_function(arg1, arg2)

OR

my_function(arg1, arg2)

3] Parameters and Arguments: Parameters are the variables listed within the function's parentheses during its definition, while arguments are the actual values passed to the function when it is called. Parameters receive the values of the arguments when the function is invoked.

4] Return Statement: User-defined functions can optionally return one or more values using the `return` statement. These values can then be assigned to variables or used in expressions when the function is called.

def add_numbers(a, b):
    result = a + b
    return result

5] Function Scope: Variables defined inside a function are typically local to that function, meaning they are not accessible from outside the function unless explicitly returned.

6] Function Documentation (Docstring): It's a good practice to include a docstring, a multi-line string enclosed in triple-quotes, to provide a description of what the function does, any parameters it takes, and what it returns.

def my_function(parameter1, parameter2):
    """
    This is the function's docstring.
    It describes what the function does and its parameters.
    """
    # Function body

7] Function Reusability: User-defined functions can be reused throughout your program or even in other programs, making them a valuable tool for organizing and modularizing code.

Here's a simple example of a user-defined function in Python:

def greet(name):
    """This function greets the person passed in as 'name'."""
    print(f"Hello, {name}!")

# Calling the function
greet("Pandru") # Output: Hello, Alice!
