Python Variables

What is Variables

In Python, variables are used to store and represent data. A variable is essentially a name that refers to a value, allowing you to access and manipulate that value throughout your code. Unlike some other programming languages, Python is dynamically typed, which means that you don't need to declare the type of a variable explicitly. Instead, the variable's type is determined at runtime based on the value it holds.

# Assigning integer value 42 to the variable 'age'
age = 42

# Assigning string value "John" to the variable 'name'
name = "John"

Valid and Invalid

It is always advisable to keep variable names descriptive and to follow a set of conventions while creating variables:

1] Variable name can only contain alpha-numeric characters and underscores (A-z, 0-9, and _ )
2] Variable name must start with a letter or the underscore character.
3] Variables are case sensitive.
4] Variable name cannot start with a number.

Example
#valid variable name
Color = "yellow"

#valid variable name
cOlor = "red"

#valid variable name
_color = "blue"

#invalid variable name
5color = "green"

#invalid variable name
$color = "orange"

Scope of variable:

The scope of the variable is the area within which the variable has been created. Based on this a variable can either have a local scope or a global scope.

Local Variable:

A local variable is created within a function and can be only used inside that function. Such a variable has a local scope.

Global Variable:

A global variable is created in the main body of the code and can be used anywhere within the code. Such a variable has a global scope.
