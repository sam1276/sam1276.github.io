Question Number 7

Section A 1 Marks

Which of the following will delete key-value pair for key = “Red” from a dictionary D1?

a. delete D1("Red")

b. del D1["Red"]

c. del.D1["Red"]

d. D1.del["Red"]

Answer
b. del D1["Red"]
Explanation

The correct way to delete a key-value pair from a dictionary D1 with the key "Red" is by using the del keyword followed by the dictionary name and the key in square brackets: del D1["Red"]. This will remove the key "Red" along with its associated value from the dictionary D1.
