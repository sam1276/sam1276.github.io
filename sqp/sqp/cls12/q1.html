Question Number 1

Section A 1 Marks

State True or False:
“In a Python program, if a break statement is given in a nested loop, it terminates the execution of all loops in one go.”

False.

In Python, if a `break` statement is given in a nested loop (a loop inside another loop), it will only terminate the innermost loop in which the `break` statement is placed. The outer loop(s) will continue their execution as usual.
