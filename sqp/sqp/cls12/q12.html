Question Number 12

Section A 1 Marks

Consider the code given below:

b = 100
def test(a):
    ______________ # missing statement
    b=b+a
    print(a,b)

Which of the following statements should be given in the blank for #Missing Statement, if the output produced is 110?

a. global a
b. global b=100
c. global b
d. global a=100

c. global b

In the given code, the function test(a) is defined to take an argument a. However, inside the function, the variable b is being used as well, and it is not defined locally within the function.

To modify the global variable b inside the function test(a), we need to declare it as a global variable using the global keyword. By adding the statement global b in the blank, we inform the function that b refers to the global variable defined outside the function.

Here's the complete code:

b = 100
def test(a):
    global b # missing statement
    b=b+a
    print(a,b)

With the global b statement, the function will recognize that b is the global variable, and when test(10) is called, it will update the global variable b by adding 10 to its current value of 100, resulting in b becoming 110, and the function will print the output as 10 110.
