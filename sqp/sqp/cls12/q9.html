Which of the following statement(s) would give an error during execution of the following code?

tup = (20,30,40,50,80,79)
print(tup)                           #Statement 1
print(tup[3]+50)                     #Statement 2
print(max(tup))                      #Statement 3
tup[4]=80                            #Statement 4

a. Statement 1

b. Statement 2

c. Statement 3

d. Statement 4

Answer
d. Statement 4

Statement 1 (tup = (20,30,40,50,80,79)): This statement defines a tuple tup containing the values (20, 30, 40, 50, 80, 79). It is a valid statement.

Statement 2 (print(tup[3]+50)): This statement accesses the element at index 3 in the tuple tup, which is 50, and then adds 50 to it, resulting in 100. It is a valid statement.

Statement 3 (print(max(tup))): This statement uses the max() function to find the maximum value in the tuple tup, which is 80, and then prints it. It is a valid statement.

Statement 4 (tup[4]=80): This statement tries to assign a new value (80) to the element at index 4 in the tuple tup. However, tuples are immutable in Python, which means their elements cannot be changed after creation. Therefore, this statement will raise an error during execution.
br So, the statement that would give an error during execution is d. Statement 4.
